# Онлайн-курсы — Angular проект

Небольшой веб-проект на Angular в формате сайта онлайн-курсов.

---

## Требования

- **Node.js** (вместе с ним устанавливается **npm**): https://nodejs.org/

---

## Установка и запуск

В корне проекта выполните:

```bash
# Установка зависимостей
npm install

# Запуск проекта
npm run start

# Веб-приложение будет доступно по ссылке:
# http://localhost:4200/
